Files & directories Navigation ‚Ä¢ Creation ‚Ä¢ Viewing

These commands help you move around the file system and work with files/folders.

Command Example Description
pwd pwd Show current working directory.
ls ls -l List files in current directory. -l = long format.
cd cd /etc Change directory to /etc.
cd .. cd .. Go one level up (parent directory).
mkdir mkdir projects Create a new directory projects.
rmdir rmdir testdir Remove empty directory testdir.
rm rm file.txt Delete file file.txt. Careful!
rm -r rm -r old_logs Delete folder and contents recursively.
cp cp file1.txt backup.txt Copy file from source to destination.
cp -r cp -r dir1 dir_backup Copy directory with all files.
mv mv old.txt new.txt Move or rename a file or directory.
touch touch notes.txt Create an empty file (or update timestamp).
cat cat file.txt Display content of a file on screen.
less less /etc/passwd View file page by page (use q to quit).
head head -n 10 file.txt Show first 10 lines of a file.
tail tail -f /var/log/messages Show last lines of file, -f = follow live updates.

Tip: Use ls -a to also see hidden files (starting with .).

Permissions & ownership Security ‚Ä¢ Access control

Linux uses permissions to control who can read, write or execute files.

Command Example Description
ls -l ls -l script.sh Show permissions, owner, group and size of file.
chmod chmod +x script.sh Add execute permission to script (make it runnable).
chmod chmod 755 script.sh Set numeric permissions (owner/group/others).
chown sudo chown user1 file.txt Change owner of file to user1.
sudo sudo mkdir /secure Run command with root/administrator privileges.

Remember: Use sudo only when needed. Wrong commands as root can break the system.

System, processes & help Monitoring ‚Ä¢ Help

Check system information, running processes and get help for commands.

Command Example Description
whoami whoami Show current logged-in user name.
uname uname -a Show system/kernel information.
df -h df -h Show disk usage of file systems (human-readable).
du -sh du -sh /var/log Show size of a directory.
ps ps aux List running processes.
top top Interactive view of CPU/memory usage (press q to quit).
free -h free -h Show memory (RAM) usage.
history history | tail Show list of previously used commands.
clear clear Clear terminal screen.
man man ls Show manual/help page of a command (press q to quit).
shutdown sudo shutdown -h now Shutdown the system immediately.
reboot sudo reboot Restart the system.

Basic networking commands Check connectivity

Command Example Description
ip ip a Show IP addresses of interfaces.
ping ping google.com Check network connectivity to a host.
curl curl http://example.com Fetch URL content from terminal.
wget wget http://file.iso Download files from the internet.
ss ss -tulpn Show listening ports and sockets.
